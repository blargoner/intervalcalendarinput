Class YAHOO.BLARGON.widget.IntervalCalendarInput - extends YAHOO.util.Element

IntervalCalendarInput instantiates an IntervalCalendar and associates it with two specified input fields which contain the lower and upper dates of the interval. The states of the IntervalCalendar and text inputs are kept in sync as the user interacts.

The format of the date strings in the input fields depends on the localization settings of the IntervalCalendar instance.
